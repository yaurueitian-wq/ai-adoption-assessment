# AI導入盤點系統

線上網址：**https://yaurueitian-wq.github.io/ai-adoption-assessment/**

用 10 道情境題自評組織的AI導入成熟度，架構改編自 Anthropic Claude Code 產品團隊負責人 Boris Cherny 於 2026/7/16 發表的《Steps of AI Adoption》。

- 5個構面各自計分（0-4），整體評級取最低分（木桶理論）
- 結果頁提供構面剖面雷達圖、落差警示、綜合診斷與優先行動建議

## 更新網站

1. 編輯本機的 `盤點系統_v2.html`
2. 複製覆蓋成 `index.html`
3. `git add index.html && git commit -m "..." && git push`

推送後約1分鐘，GitHub Pages會自動更新。
